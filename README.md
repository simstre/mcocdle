# MCOCdle

A daily guessing game for Marvel Contest of Champions. Guess the mystery champion each day based on attribute clues — class, gender, size, alignment, affiliation, fighting style, and release year.

## How It Works

- A new champion is selected every day at 8am PST
- Each guess reveals how the champion's attributes compare to the target
- Green = exact match, yellow = partial, red = no match, arrows = higher/lower
- After 10 guesses, a hint system reveals the least-matched attribute and suggests 5 possible champions
- Share your results with an emoji grid showing your guessing path
- Leaderboard tracks the first solvers each day
